# HAR-Using-Machine-Learning-with-end-to-end-deployment-pipeline
This project presents a complete Human Activity Recognition (HAR) pipeline using smartphone accelerometer data, covering every stage from exploratory data analysis to real-time mobile-ready inference. The goal is to accurately classify daily human activities such as Walking, Sitting, Standing, Jogging, Lying Down, and Stairs  using machine Learning
# Dataset Link- https://www.cis.fordham.edu/wisdm/dataset.php
